Click a link above to view a webpage created for this project.

This index lists and links the webpages created for our final databse application project in CS3430. With this database application, a pokemon trainer can manage and track their pokemon and see how they compare with other trainers in the system. A pokedex is provided, as is a directory of gyms and badges. A trainer can create an account, add their pokemon, update them, view them, and if one should so wish, release them.
